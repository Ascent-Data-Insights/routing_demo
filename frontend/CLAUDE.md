# Frontend Specification

The frontend will be a React app - using tailwind and Tailwind plus for styling. 

Tailwind Plus components are inside of this directory for use. 

## Branding

Branding will use the Ascent Data Insights logo, fonts and color palette, however it should be easy to change these to a potenial clients' font, colors, and logo. 

**Ascents' Fonts (Google Fonts): **
Kodchasan for Headers
Manrope for Body text

**Ascents' colors: **
- Primary: `#03344E` (Deep Blue)
- Secondary: `#4785BF` (Sky Blue)
- Accent: `#FB8500` (Orange)

**Ascent's logo:** At logo.png within this project.
